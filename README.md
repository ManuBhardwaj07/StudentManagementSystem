A simple Java-based application to manage student records efficiently. This project allows users to perform CRUD operations (Create, Read, Update, Delete) on student data.

📌 Features
Add new student records
View all students
Search student by ID
Update student details
Delete student records
File storage / console-based interaction (depending on your implementation)
🛠️ Tech Stack
Language: Java
Concepts: OOP, Collections, File Handling
IDE: VS Code / IntelliJ / Eclipse
